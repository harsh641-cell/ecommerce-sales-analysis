# E-Commerce Sales Analysis

## Project Overview

This project analyzes an E-Commerce sales dataset using Python to discover business insights, customer purchasing behavior, payment preferences, sales trends, and product performance.

The analysis includes data cleaning, exploratory data analysis (EDA), and data visualization using Pandas, Matplotlib, and Seaborn.

---

## Objectives

- Clean and prepare the dataset
- Analyze customer purchasing patterns
- Find the most popular payment methods
- Identify top-selling product categories
- Analyze revenue distribution
- Visualize important business insights
- Practice real-world data analysis techniques

---
